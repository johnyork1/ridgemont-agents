Section 4: How to Connect (No More Bouncing)

  1. Open claude.ai/settings/connectors in your browser (or Claude Desktop → Settings → Connectors)
  2. Search for the connector by name (e.g., "NPI Registry", "Notion", "Figma")
  3. Click Connect and complete the OAuth/authorization flow
  4. Repeat for all connectors you want — do them all in one batch
  5. Return to any Cowork session and say "I connected [names]" — the Master Agent will embed the references in the appropriate agent config files
Tip: Connectors are account-level — once connected, they're available to ALL Cowork sessions immediately. You never need to reconnect them per-agent.
