# Frozen Cloud Website

Static single-page artist landing page for the Frozen Cloud music project, hosted on GitHub Pages.

## What It Does

- Dark-themed landing page with ice-blue brand palette
- Glassmorphism design using Google Fonts (Orbitron + Inter)
- All HTML/CSS/JS contained in a single `index.html` file
- No frameworks or build tools required

## Usage

Open `index.html` in a browser for local preview. Deploy by pushing to GitHub (GitHub Pages).

To update content, edit `index.html` directly.

| Name | Hex |

## Requirements

- None (static HTML). Any web browser works for local testing.
- GitHub Pages (or any static host) for deployment.
