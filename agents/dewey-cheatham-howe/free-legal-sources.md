# Free Legal Data Sources — Quick Reference

## Case Law Databases

| Source | URL | Coverage | API? |
|--------|-----|----------|------|
| CourtListener | courtlistener.com | Federal + state courts (3,352 jurisdictions) | Yes — REST API v4.3 (free token) |
| Google Scholar Case Law | scholar.google.com | Federal + state case law | No (web scraping only) |
| Justia | justia.com | Federal + state case law, codes, regulations | No |
| Caselaw Access Project | case.law | All official U.S. case law through 2020 | Via CourtListener |
| FindLaw | findlaw.com | Federal + state case law | No |

## MCP Servers for AI Integration

## API Keys Needed (All Free)
- **CourtListener**: Free account at courtlistener.com/sign-in/ → API token in settings
- **Google Scholar MCP**: No API key needed (web scraping)
- **All other sources**: No authentication required for web access
